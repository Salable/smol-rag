# Cancel a subscription

You can cancel a subscription with its associated `subscriptionUuid`.

<Tabs>
<TabItem value="node" label="Node.js">

:::info

The second argument to `subscriptions.cancel` represents when you want the
subscription to cancel.

| When  | Description                                                |
| ----- | ---------------------------------------------------------- |
| `now` | Immediately cancels the Subscription                       |
| `end` | Cancels the Subscription at the end of it's billing period |

:::

```js
import { Salable } from '@salable/node-sdk';

const salable = new Salable('your-salable-api-key', 'v2');
await salable.subscriptions.cancel('your-subscription-uuid', { when: 'end' });
```

</TabItem>

<TabItem value="curl" label="cURL">

:::info

The `when` query parameter represents when you want the subscription to cancel.

| When  | Description                                                |
| ----- | ---------------------------------------------------------- |
| `now` | Immediately cancels the Subscription                       |
| `end` | Cancels the Subscription at the end of it's billing period |

:::

```bash
curl
  -XPUT
  -H 'x-api-key: your-salable-api-key'
  -H 'version: v2'
  'https://api.salable.app/subscriptions/{subscriptionUuid}/cancel?when=end'
```

</TabItem>
